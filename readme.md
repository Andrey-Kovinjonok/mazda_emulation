# mazda_emulation

Instruction :

1) Copy one or more *_reinstall.up file (exemple : cmu150_NA_55.00.753A_reinstall.up )
2) Call extract.sh
3) call node server.js
4) open .../jci/gui/index.html in chrome (exemple : cmu150_NA_55.00.753A/jci/gui/index.html )

keys
m, n = cw and ccw
arrow = joystick
del   = back
enter = select
